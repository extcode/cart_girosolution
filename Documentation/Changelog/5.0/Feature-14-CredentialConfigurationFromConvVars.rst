========================================================================
Feature: #14 - Credential Configuration from $GLOBALS['TYPO3_CONF_VARS']
========================================================================

Description
===========

It is now also possible to configure the credentials in `$GLOBALS['TYPO3_CONF_VARS']` and load there the
passwords from ENV variables.

For more informations go to :ref:`Credentials from $GLOBALS['TYPO3_CONF_VARS'] <credentials-from-confvars>`
